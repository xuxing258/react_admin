### 简介

> 使用的react - antd - node - mongoodb 创建的后台管理系统

- root文件夹前端页面
- react_node搭建node后端

### 启动

```js
// node文件夹 
npm install //安装
node app.js

// 前端页面
npm install   // 安装
npm run start // 启动
```
